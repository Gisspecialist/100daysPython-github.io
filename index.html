Help / User Guide

Search, expand sections, load examples, or start a guided walkthrough.

This app organizes Python lessons by day and by sheet. Each sheet includes Concept, Day, Filename, 18 numbered code lines, Comments, Summary, and Lecture.

  1. Select or add a day.
  2. Create one or more sheets for that day.
  3. Enter code in lines 01–18.
  4. Load the code into the runner and run it.
  5. Export the full notebook JSON for backup.

Use Select Day to switch days. Use New Sheet for Selected Day to create another sheet under the same day. Previous Sheet and Next Sheet move between sheets for the selected day.

Autosave stores your notebook in browser local storage. Export Full Notebook JSON downloads one file containing all days and sheets. Import JSON restores a saved notebook file.

The runner uses Pyodide in the browser and shows an online-style transcript: Run started, Initializing environment, Installing packages, Running code, and then your output.

When code uses input(), answer directly in the Output panel and press Enter. No popup prompt is used.

Click an example to load it into the Python Runner.

  • If Pyodide is not ready, wait for the runtime status to say it is ready.
  • If packages fail to install, check the package name and internet connection.
  • If code does not run, check the terminal output for the Python traceback.
  • Export JSON regularly so your notebook has a backup copy.
